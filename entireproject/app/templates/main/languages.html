Javascript

JavaScript, ECMAScript spesifikasiyasına uyğun şərh edilmiş bir proqramlaşdırma dilidir.
JavaScript yüksək səviyyəli, vaxtında tərtib edilmiş və çox paradiqmadır.
Buruq-mötərizəli sintaksis, dinamik yazma, prototip əsaslı obyekt yönümlü
və birinci dərəcəli funksiyalara malikdir. Bu yolda JavaScript əsaslarını, eləcə də vədlər,
asinxron proqramlaşdırma, düşüncə kimi daha inkişaf etmiş mövzuları öyrənəcəksiniz.

Angular

Angular, HTML-də dinamik və interaktiv tətbiqetmələr yaratmaq üçün tam bir JavaScript çərçivəsidir.
Vebdəki ən isti çərçivələrdən biri olmaqdan başqa, Angular öyrənmək asandır,
eyni zamanda kompleks bir səhifəlik veb tətbiqetmələr hazırlamağa kömək edəcək qədər güclüdür.
Bu yol, Angular 2 və daha çoxunu əhatə edən məzmunu əhatə edir. Orijinal versiya üçün AngularJS yolumuza baxın.
